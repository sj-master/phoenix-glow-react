GlowInput

Inputs

A text field with a neon focus ring. Focus it to see the cyan glow — type to see the live value.

You typed: — 
<GlowInput
  value={value}
  onChange={(e) => setValue(e.target.value)}
  placeholder="Type something glowing…"
/>
GlowCard

Cards

Neon-bordered panels that lift and glow on hover. Each takes an accent color. Hover them.

Purple

Hover for the neon lift.

Cyan

Set the accent per card.

Green

hoverGlow can be disabled.

<GlowCard accent="cyan">
  <h3>Glowing card</h3>
  <p>Hover me.</p>
</GlowCard>

GlowAccordion

Accordion

Collapsible panels with a glowing open state. Click a header.

A React component library wrapping the Phoenix Glow neon aesthetic.
Run npm install phoenix-glow-react and import the CSS once.
Components use semantic roles and ARIA attributes where relevant.
<GlowAccordion title="Details" defaultOpen>
  Hidden content goes here.
</GlowAccordion>
